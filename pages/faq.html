Frequently Asked Questions

Find answers to common questions below

Where can I see recently added items? ▼
Use the panels on the dashboard to view the most recent items added to the system. Updates appear automatically.
What should I do if something isn't working? ▼
Try refreshing the page first. If the problem persists, submit a ticket using the Support Ticket page, refer to the Common Questions/Issues shelf on the wiki, or contact support directly.
How do I submit a ticket? ▼
You can submit tickets via the Ticket Submission page. Each ticket will be handled promptly by the support team.
I am interested in setting up one of the servers for my own use, how can I? ▼
Check out the Getting Started section on the Wiki! It has various support related topics created including but not limited to: clients, setting up the service, and the use of services.
How can I make a request for a show/movie? ▼
Visit the media request service, sign in with your Jellyfin credentials, then search and request the wanted media or you can make use of the Request tab on the main Jellyfin site or through the Streamyfin client via the Discover section in search.
How can I make a request for music? ▼
The easiest way to get more music added to the server is by using the Playlist Importer Tool. It will scan the playlist provided, create one with available tracks on Jellyfin, search for missing tracks on Lidarr, then download and import them on a scheduled basis. It will update the playlist every 6 hours for the following 24 hours with newly found tracks. If no new songs are available before it is done downloading through deemix, it will download the individual songs directly through spotify through Spooty.
The naming seems a bit confusing - how are the services' domains named? ▼
All of the domains/URLs that are used for each service are sub-domains of your configured domain. For example, if your domain is example.com, the wiki would be at wiki.example.com. The sole exception is the main Jellyfin service if it is tied to the root domain itself.
